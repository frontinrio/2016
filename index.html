  • Nathalia Bruno

    Acessibilidade: Quem se importa?

    10h00 - 10h30

    Afinal, o que é acessibilidade? O quanto isso impacta no meu site? Por que eu deveria me importar? Se desenvolvemos para navegadores usados por 0.35% de nossa população, porque não pensar nos 10% com algum tipo de necessidade especial? Esta palestra visa explicar como, quando, onde e porque aplicar as regras de acessibilidade no seu desenvolvimento. E o que perdemos quando ignoramos as regras.

  • Maujor

    O elemento picture para imagens responsivas

    11h30 - 12h15

    O elemento picture cria um padrão de marcação HTML que possibilita ao desenvolvedor declarar um conjunto de imagens. O uso de media queries regem as condições para que o navegador escolha a imagem mais apropriada a ser servida.

  • Guto Xavier

    Revisitando Local Storage

    14h00 - 14h30

    A palestra tem por objetivo apresentar e relembrar possibilidades da utilização da Web Storage API com o armazenamento local do lado do cliente, quando por exemplo a conexão com internet é perdida. podendo persistir dados armazenados quando conexões forem restabelecidas.

  • Rafael Fragoso

    Web Bluetooth API

    14h30 - 15h00

    Em uma das diversas iniciativas do Google em direção a Web Física, foi feito um esboço da Web Bluetooth API para permitir que páginas web se comuniquem diretamente com dispositivos Bluetooth, abrindo um novo leque de possibilidades para a web. Nessa talk eu vou explicar sobre o funcionamento do bluetooth, seus impactos no mundo tecnológico e falarei sobre essa API para web, com um possível demo.
